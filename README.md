  <h2>🐔 O que é o GRIOT - Memória e História Afro-Brasileira</h2>
  <p>
     GRIOT é uma aplicação web interativo que reúne REDs (recursos educacionais digitais) multimídia sobre cultura negra. Projetado como uma experiência imersiva, o site convida
     visitantes a percorrer as mais diversas produções culturais afro-brasileiras, como artes plásticas, obras literárias e obras musiciais. GRIOT fornece acesso à ancestralidade negra do Brasil e possibilita a comparação com a contemporaneidade, abrangendo os repertórios educaionais. O projeto do teve início no mês de abril de 2025.
  </p>
</div>

<div>
  <h2>🚫 Problema</h2>
  <p>
    Após a promulgação da Lei nº 10.639/2003, que estabelece a obrigatoriedade do ensino da História e Cultura Afro-brasileira na educação básica, observou-se uma lacuna na disponibilidade de materiais didático-pedagógicos adequados à temática étnico-racial. Tal cenário relaciona-se à permanência de abordagens eurocêntricas nos currículos escolares, que limitam a representação da população negra e contribuem para a reprodução de estereótipos. Estas práticas pedagógicas surgem de modelos políticos que valorizam aspectos culturais europeus, adotando um unilateralismo histórico em detrimento da história africana do país. A veiculação e uso deste tipo de material em sala de aula resulta, portanto, no efeito contrário do que a lei propõe combater.
  </p>
</div>

<div>
  <h2>🧱 Contrução da ideia</h2>
  <p>
     Identificada esta problemática, desenvolveu-se a aplicação web GRIOT, que funciona como um acervo para materiais culturais relacionados à cultura afro-brasileira. Para arquitetar esta plataforma estão sendo usados o HyperText Markup Language (HTML), JavaScript e Cascading Style Sheets (CSS) para estruturação, interação e estilização, o Hypertext Preprocessor (PHP) para processamento de comandos lógicos e o banco de dados Supabase para persistência de dados na nuvem. Para versionamento e hospedagem de código foram utilizados o Git e Github, sendo o software de edição de código o Visual Studio Code.

<div>
  <h2>📝 Método</h2>
  <p>
     O método para selecionar obras que povoam o sistema consiste em: 1 - Procura e seleção livre das obras; 2 - Curadoria da obra em relação à licença de uso; 3 - Levantamento de dados sobre a obra para contextualização no acervo. A plataforma GRIOT busca ainda servir de espaço para valorização de produtores de cultura afro-brasileira da Região Metropolitana de Curitiba, garantindo representatividade local com as devidas burocracias legais em relação a licenças de uso. Desta maneira, a aplicação entrega recursos educacionais digitais com descrições completas e sem inconsistências quanto ao uso de mídias.
    A plataforma contempla seções como pinturas, fotografias, músicas, literatura, produções audiovisuais, legislação e linha do tempo, organizadas de maneira responsiva.
  </p>
</div>

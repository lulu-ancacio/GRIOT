  <h2>O que é o GRIOT - Memória e História Afro-Brasileira</h2>
  <p>
     GRIOT é uma aplicação web interativo que reúne REDs (recursos educacionais digitais) multimídia sobre cultura negra. Projetado como uma experiência imersiva, o site convida
     visitantes a percorrer as mais diversas produções culturais afro-brasileiras, como artes plásticas, obras literárias e obras musiciais. GRIOT fornece acesso à ancestralidade negra do Brasil e possibilita a comparação com a contemporaneidade, abrangendo os repertórios educaionais.
  </p>
